# Ahmed Eid Website — Stage 1

واجهة المرحلة الأولى لموقع أحمد عيد الشخصي.

## ما تم تنفيذه
- تنظيف الـ branding القديم.
- تحسين الصفحة الرئيسية ورسائل البيع.
- إضافة صفحات: الخدمات، دراسات الحالة، التواصل.
- تحسين SEO الأساسي عبر metadata و sitemap و robots.
- تجهيز المشروع للمرحلة الثانية (لوحة تحكم + قاعدة بيانات).

## التشغيل محليًا
```bash
npm install
npm run dev
```

## البناء للإنتاج
```bash
npm run build
npm run start
```

## ملاحظات
- رقم الواتساب ما زال Placeholder داخل صفحة التواصل ويحتاج استبدالًا.
- المرحلة الثانية ستضيف Payload CMS و PostgreSQL ولوحة تحكم.
